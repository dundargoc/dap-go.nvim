# dap-go.nvim

Golang with Delve over Debug Adapter Protocol for [Neovim](https://neovim.io/) (>= 0.5)

# What is DAP Go

`dap-go.nvim` os an extension for [nvim-dap](https://github.com/mfussenegger/nvim-dap) to configurate Go with Delve debugger.
